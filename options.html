Window Names

Show window names in the OS title bar

Firefox can show each window's name in a title bar across the top of the window — handy on macOS, where the title bar is hidden by default. Firefox doesn't let an extension flip this for you, but it's a one-time toggle:

  1. Right-click an empty area of the toolbar.
  2. Choose Customize Toolbar…
  3. At the bottom-left, tick the Title Bar checkbox.
  4. Click Done.

If that didn't open, do it manually with the steps above.

Where names show up

  • The toolbar button badge and tooltip (always visible).
  • The OS title bar (once enabled above).
  • The taskbar / window switcher.
  • The macOS dock icon's window list.
